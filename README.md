 HR Attrition Dashboard (Power BI)

This project focuses on analyzing employee attrition to understand the key factors influencing why employees leave an organization. The dashboard provides insights into workforce demographics, attrition distribution across departments, job roles, salary levels, and performance ratings.

Objective
To help HR and management identify:
- Which employee groups are most likely to leave
- Key parameters driving attrition (age, salary slab, department, job role, etc.)
- Areas to improve employee retention and engagement

 Tools & Technologies Used
| Tool/Concept | Purpose |
|-------------|---------|
| Power BI | Data Modeling & Dashboard Creation |
| Power Query | Data Cleaning & Transformation |
| DAX (Data Analysis Expressions) | Calculated Columns & Measures |
| Excel / CSV | Data Source |

 Project Workflow
1. **Data Cleaning**: Removed null values, standardized categorical labels.
2. **Data Modeling**: Created relationships between employee datasets.
3. **DAX Measures**:
   - Total Employees
   - Attrition Count
   - Attrition Rate
   - Average Salary
4. **Visualization Development**: Designed dynamic visuals & filters.

 Dashboard Highlights
- **Overall Attrition Rate** visualization
- Attrition by:
  - Age Group
  - Gender
  - Department
  - Education Level
  - Job Role
  - Salary Slab
- Employee Count & Average Monthly Income KPIs
